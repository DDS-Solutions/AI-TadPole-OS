## Why Tadpole OS?

Tadpole OS is built for operators who demand total control over their AI infrastructure. Unlike cloud-based orchestrators, the Tadpole engine runs locally, persists memory across sessions, and executes complex multi-agent workflows with strict budget and security barriers.

| Pillar | Focus | Technology |
| :--- | :--- | :--- |
| **Intelligence** | Multi-Model Slots | GPT-4, Gemini, Ollama |
| **Orchestration** | Recursive Scaling | Multi-threading (Tokio) |
| **Memory** | Institutional Wisdom | Vector DB (LanceDB) |
| **Observability** | Swarm Pulse | Binary WebSockets |
